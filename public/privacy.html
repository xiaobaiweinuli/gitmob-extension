GitMob Extension 隐私政策

最后更新：2025 年

GitMob Extension（以下简称「插件」）是一款用于 GitHub 收藏夹跨设备同步的浏览器扩展。我们高度重视用户隐私，本政策说明插件如何处理您的数据。

1. 收集的数据

  • GitHub Personal Access Token（PAT）：您手动输入并仅存储在本地（chrome.storage.local），不会上传至任何服务器。PAT 用于调用 GitHub API 验证您的身份及获取仓库信息。
  • GitHub 用户信息：通过 PAT 获取的 login（用户名）和头像 URL，用于界面显示及云端数据隔离的唯一标识（user_id）。
  • 收藏夹数据：分组名称、分组描述、仓库名称（owner/repo）及仓库元信息（描述、语言、Star 数等），存储在您选择的同步服务（默认为 sync.gitmob.xyz）的 Cloudflare D1 数据库中。
  • 设备 ID：本地生成的随机 UUID，用于多设备冲突检测，不包含任何个人信息。

2. 数据使用方式

  • PAT 仅用于向 GitHub API 验证身份，不会存储在任何远程服务器上。
  • 收藏数据仅用于在您的设备间同步，不会用于广告、分析或任何第三方用途。
  • 同步服务按 user_id（GitHub login）严格隔离，其他用户无法访问您的数据。

3. 数据存储

  • 本地：PAT、设备 ID、收藏数据缓存存储于浏览器的 chrome.storage.local，不与其他网站共享。
  • 云端：收藏数据存储在您选择的同步服务（默认为由 Cloudflare 托管的 sync.gitmob.xyz）。您也可以部署自己的同步服务以完全掌控数据。

4. 数据删除

删除插件将自动清除所有本地存储的数据（包括 PAT）。云端数据将在您停止使用后自然过期，您也可以通过自托管同步服务的管理界面手动删除。

5. 第三方服务

  • GitHub API：用于验证 PAT 和获取仓库信息，受 GitHub 隐私政策约束。
  • Cloudflare：默认同步服务的基础设施提供商，受 Cloudflare 隐私政策约束。

6. 联系我们

如有隐私相关问题，请通过 GitHub Issues 联系我们。
